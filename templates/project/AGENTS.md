# Project Agent Rules

This is a Hive OS project workspace.

## Rules

- Read `wiki/index.md`, `wiki/log.md`, and `wiki/status.md` before project work.
- Keep durable decisions in `wiki/decisions.md`.
- Keep task work inside `tasks/<task-slug>/`.
- Put deliverables in `outputs/` or task-specific `outputs/`.
- Do not access other project workspaces unless invited.
- Do not assume environment admin is a project member.
